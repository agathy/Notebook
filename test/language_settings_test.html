🔧 语言设置功能测试

🎯 测试目标：验证在单词列表页面点击语言设置按钮是否能直接显示语言设置界面，而不是跳转到首页
步骤 1: 准备测试环境

首先确保有一些测试数据，然后访问单词列表页面

操作步骤：
  1. 点击下面的链接打开首页
  2. 如果没有设置语言，先设置母语和学习语言
  3. 添加一些测试单词（可选，但建议添加）
  4. 然后点击"我的单词本"进入单词列表页面
🏠 打开首页
步骤 2: 测试语言设置功能

在单词列表页面测试语言设置按钮的功能

操作步骤：
  1. 点击下面的链接打开单词列表页面
  2. 找到右上角的设置按钮（齿轮图标）
  3. 点击设置按钮
  4. 验证：应该直接显示语言设置界面，而不是跳转到首页
📚 打开单词列表页面
步骤 3: 测试设置修改功能

测试在单词列表页面修改语言设置的完整流程

操作步骤：
  1. 在语言设置界面中，尝试修改母语或学习语言
  2. 点击"保存设置"按钮
  3. 验证：应该返回单词列表页面
  4. 验证：顶部的语言显示应该更新
  5. 验证：如果切换了母语，应该看到数据迁移的提示
  6. 验证：表格头部应该反映新的语言设置
步骤 4: 测试取消功能

测试取消按钮是否正常工作

操作步骤：
  1. 再次点击设置按钮进入语言设置
  2. 修改一些设置（但不要保存）
  3. 点击"取消"按钮
  4. 验证：应该返回单词列表页面
  5. 验证：语言设置应该没有改变
步骤 5: 对比测试（修复前 vs 修复后）
✅ 修复后的预期行为：
  • 点击设置按钮 → 直接显示语言设置界面
  • 修改设置后 → 返回单词列表页面
  • 界面更新 → 实时反映新的语言设置
  • 数据迁移 → 自动处理语言切换
❌ 修复前的问题行为：
  • 点击设置按钮 → 跳转到首页
  • 需要在首页设置 → 额外的页面跳转
  • 用户体验差 → 不直观，操作繁琐
