# 卡片堆滑动功能演示

## 功能概述

已成功实现卡片堆叠视图，所有单词卡片堆叠在一起，支持多种交互方式：

### 🎯 主要特性

1. **卡片堆叠效果**
   - 所有卡片堆叠在一起，形成3D层次感
   - 顶部卡片完全显示，后面的卡片逐渐缩小和透明
   - 最多显示4张卡片的堆叠效果

2. **滑动交互**
   - 👆 **触摸滑动**：在移动设备上支持触摸滑动
   - 🖱️ **鼠标拖拽**：在桌面设备上支持鼠标拖拽
   - ⬅️ **左滑**：切换到上一张卡片
   - ➡️ **右滑**：切换到下一张卡片

3. **视觉反馈**
   - 滑动时卡片会跟随手指/鼠标移动
   - 卡片会根据滑动方向旋转
   - 实时显示滑动提示（"👈 上一个" / "👉 下一个"）
   - 滑动阈值达到时自动切换卡片

4. **控制按钮**
   - 上一个/下一个按钮
   - 卡片计数器显示当前位置
   - 按钮状态自动更新（首尾禁用）

### 🎨 动画效果

- **卡片切换**：流畅的滑入滑出动画
- **堆叠更新**：卡片位置和大小的平滑过渡
- **触摸反馈**：容器边框光效提示
- **回弹效果**：未达到阈值时卡片回到原位

### 📱 响应式设计

- 桌面设备：350x500px 卡片尺寸
- 移动设备：300x450px 卡片尺寸
- 控制按钮在移动设备上隐藏文字，只显示图标

## 使用方法

1. 在单词列表页面点击"卡片堆"视图按钮
2. 使用以下方式切换卡片：
   - 在卡片上左右滑动/拖拽
   - 点击"上一个"/"下一个"按钮
   - 键盘方向键（如果实现）

## 技术实现

### 核心模块
- `CardStackManager` 类：管理卡片堆的所有交互逻辑
- 支持触摸和鼠标事件
- 模块化设计，易于扩展和维护

### CSS 特性
- 3D 变换和透视效果
- 流畅的过渡动画
- 响应式布局
- 现代化的视觉设计

### 兼容性
- 支持现代浏览器
- 移动设备触摸优化
- 桌面设备鼠标交互
